#PIC16F1619

Overview
--------------------------------------------
* Description: 

A set of projects and libraries for a **Microchip** 
PIC 16F1619 Micro-controller 20-pin, 8-bit.

* Author: Gavin Lyons 
* Toolchain: XC8 , MPLAB X IDE and MCC.
